# Nexus News App
A modern Android app built with Kotlin, Jetpack Compose and Retrofit.

## Features
- Browse latest news worlwide
- Search specific articles
- Bookmark favourite articles
- Room database for offline caching
- Category filter using content type and geological locations (continents)
- About us with Email and Instagram contacts

## Tech Stack
-Kotlin
- Jetpack Compose for UI
- Retrofit for API calls
- Room for local storage
- Coroutines for async operations
- ViewModel and LiveData
- Hilt for dependency injection

## API
The app uses 'NewsData.io'(https://newsdata.io/) API

## Setup
i. Get your API key from official NewsData.oi website
ii. Add it to lacal.properties (NEWS_API_KEY = your_api_key)
